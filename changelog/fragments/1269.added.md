`IO.get_merged_data()` returns a fresh union of the input and output stores (output overrides input on overlap); `as_dict=False` returns a `DisciplineData` instead of a plain `dict`. `IO.propagate_to_input(updates)` copies entries of `updates` whose name is in the input grammar into the input store — used by MDA solvers to propagate auto-coupled outputs back to the input side between iterations.
