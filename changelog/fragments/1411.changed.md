BREAKING CHANGES:

- `Dataset.group_names`, `Dataset.variable_names`, `Dataset.variable_identifiers`, `Dataset.get_group_names` and `Dataset.get_variable_names` no longer return sorted values; they now preserve insertion order.
- For the same reason, `IODataset.input_names` and `IODataset.output_names` no longer return sorted values.
- For the same reason, `OptimizationDataset.design_variable_names`, `OptimizationDataset.constraint_names`, `OptimizationDataset.equality_constraint_names`, `OptimizationDataset.inequality_constraint_names`, `OptimizationDataset.objective_names` and `OptimizationDataset.observable_names` no longer return sorted values.
