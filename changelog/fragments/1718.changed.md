- The `Dataset.variable_names_to_n_components` attribute uses the same order as the variables in the dataset.
- The `Dataset.get_normalized` method returns a new dataset using the same order as the variables in the original dataset.
- When the `var_names` option is empty, unsupervised machine learning models uses all the variables of the dataset while keeping their order.
- When the `variable_names` option is empty, `ScatterMatrix` uses all the variables of the dataset while keeping their order.
- When the `variable_names` option is empty, `ColorEvolution` uses all the variables of the dataset while keeping their order.
- When the `variable_names` option is empty, `Lines` uses all the variables of the dataset while keeping their order.
- When the `variable_names` argument is empty, `BaseStatistics` uses all the variables of the dataset while keeping their order.
